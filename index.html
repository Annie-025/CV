CASE FILE 01

ESG APP DESIGN

围绕高校校园低碳实践，设计“碳索鸭”APP，以轻量化设计记录、量化和激励学生的日常低碳行为，形成行为记录、积分反馈、社交互动和习惯培育的完整产品闭环。

RoleProduct Thinking / UI Prototype / UX Flow
ToolsFigma / Research / Business Analysis
过程记录Research / Structure / Feasibility
Outcome形成校园低碳行为管理工具的产品结构、轻量化实现路径和运营可行性方案。

过程记录

01

RESEARCH / 背景研究

从宏观政策、环保 APP 市场和中山大学校园环保现状切入，分析绿色低碳教育、校园垃圾分类、绿色出行和环保行动持续性不足等背景问题。

02

PAIN POINTS / 痛点定义

通过对现有环保 APP 的对比，提炼出用户体验不佳、用户黏性不足、功能同质化和校园适配性弱等痛点，并进一步聚焦校园垃圾分类、绿色出行、旧物交换和光盘行动四类高频场景。

03

PRODUCT STRUCTURE / 产品结构

围绕“游戏化设计—碳友圈积分量化—个性化激励”建立产品定位，设计碳行为引导、碳行为实践、碳成果分享和碳习惯培育四大模块。

04

LIGHTWEIGHT SYSTEM / 轻量化实现

采用“轻量化—低成本—高兼容”的技术思路，用拍照上传、人工互评、积分图表、日历记录和社交互动降低实现门槛，避免过度依赖复杂 AI 或 VR 功能。

05

OPERATION & FEASIBILITY / 运营与可行性

进一步设计推广路径、线上线下渠道、用户增长预测和经济可行性模型，验证产品在试运营、全校推广和后续商业化阶段的落地可能。

06

REFLECTION / 项目沉淀

这个项目记录了我从单纯界面设计转向产品结构、用户激励机制、商业可行性和技术落地综合思考的过程。

CASE FILE 03

MAISON MARGIELA CITY SCENT

围绕 Maison Margiela 的记忆叙事与城市身份，提出 City Scent 香氛概念，将市场分析、用户画像、产品系统、定价策略和传播路径整合为完整品牌体验方案。

RoleBrand Experience / Market Analysis / Pricing Narrative
ToolsResearch / Product System / HTML / CSS
过程记录Market / Concept / Pricing / Growth Journey
Outcome形成涵盖市场洞察、产品系统、定价逻辑和传播路径的完整品牌体验提案。
查看项目

过程记录

01

MARKET ANALYSIS / 市场分析

分析香水市场规模、增长率、低渗透率与低使用频次之间的机会，提出通过更日常化、场景化的城市香氛产品提高用户使用频率。

02

USER INSIGHT / 用户洞察

建立三类用户画像：城市移居者、本地身份认同者和文化旅行者，发现他们都希望通过香味建立与城市、记忆和自我身份之间的情感连接。

03

CONCEPT DEVELOPMENT / 概念发展

提出“City Memory Science → Olfactory Identity”的核心概念，将城市记忆、个人情感和香气体验结合，形成城市专属香氛产品的差异化定位。

04

PRODUCT SYSTEM / 产品系统

设计线上限时零售与线下深度定制两类体验：线上通过情景问答推荐城市香氛，线下通过旗舰店、香氛工作坊和个性化定制增强沉浸式体验。

05

PRICING & EXPERIENCE / 定价与体验

基于不同用户支付意愿，设计 discovery set、standard 和 premium 产品线，并结合线上低风险试用与线下高端定制满足不同消费层级。

06

GROWTH JOURNEY / 传播路径

规划从 April-May 预热、June-August KOC 场景叙事与线下展览，到未来 ESG 互动、多渠道合作和城市文化基金的增长路径。

07

REFLECTION / 项目沉淀

这个项目展示了我将市场分析、品牌叙事、产品设计、定价逻辑和传播策略整合成完整商业方案的能力。
